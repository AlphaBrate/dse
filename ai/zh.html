1. 獲取憑證

要啟用 AI 引擎，你必須通過 OpenRouter 閘道生成安全的「握手金鑰」(Handshake Key)。

  • 前往 openrouter.ai 並點擊 Get API Key。
  • 點擊：
  • 使用你偏好的身份提供商進行驗證（建議使用 Google 帳號）。
  • 如果系統詢問 "Where did you first hear about us?"，請選擇 Other / Not Sure。

2. 配置金鑰參數

請參考以下參數進行設定，以確保在使用記錄中清晰顯示，並保障帳戶安全。

Create Key ✕
AlphaBrate for DSE
留空代表不設限
N/A ▼
No expiration (永不過期) ▼
  • 識別名稱： 建議將金鑰命名為 AlphaBrate for DSE。這有助於你在分析面板中追蹤使用數據。
  • 安全限制： 若你在公共電腦上操作，我們強烈建議設置具體的 Expiration (過期日期)。否則，其他選項請保持留空。
  • 最後步驟： 請立即複製生成的字符串。基於安全原因，完整的 API 金鑰只會顯示一次。

3. 本地部署

立即將憑證輸入到 AlphaBrate for DSE 應用程式中。

API KEY 密鑰
sk-or-...

將 sk-or-... 字符串粘貼至欄位內，然後選擇 Save。
